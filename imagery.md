## Scale

Defined as `fontSize.*` in `brand.json`. Do not introduce sizes outside the
scale.

| Token | Size | Used for |
| ----- | ---- | -------- |
| `3xl` | 3rem | Page hero |
| `2xl` | 2rem | Section headings |
| `xl` | 1.5rem | Sub-headings |
| `base` | 1rem | Body copy — minimum size for paragraphs |
| `sm` | 0.875rem | Captions, labels |
