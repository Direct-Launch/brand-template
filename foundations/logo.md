## Variants

```logo
variants: primary, reversed, mono, mark
```

| Variant | File | Use when |
| ------- | ---- | -------- |
| Primary | `assets/logo/primary.svg` | Default, on light backgrounds |
| Reversed | `assets/logo/reversed.svg` | On dark or photographic backgrounds |
| Mono | `assets/logo/mono.svg` | Single-colour print, embossing, engraving |
| Mark only | `assets/logo/mark.svg` | Favicon, avatars, small square spaces |

## Misuse

Never:

- Stretch, squash, or rotate
- Recolour outside the approved variants
- Add effects — shadows, outlines, gradients
- Place on a background that fails the contrast minimum
- Recreate or re-typeset the wordmark
- Use the old logo (see `assets/logo/archive/`)
